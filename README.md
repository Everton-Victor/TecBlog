# TecBlog
É uma homepage de um blog de tecnologia fictício, apenas o front-end da home.

## Objetivo
É colocar em prática meus conhecimentos de CSS3 adiquiridos dos meus estudos pessoais.

## Tecnologias usadas
### Front-End
- HTML5
- CSS3
    - Float
    - Clear
    - Display block
